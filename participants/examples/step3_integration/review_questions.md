# 리뷰 질문 템플릿
리뷰어가 바로 복사해 사용할 수 있는 질문 목록입니다.

## 핵심 질문
1. 경계: 해당 컨텍스트가 책임지는 용어가 두 컨텍스트에 동시에 등장하나요?
2. 책임: 애그리게잇 루트가 거절/승인 권한을 모두 가져야 하는 이유는 무엇인가요?
3. 트랜잭션: 상태 전이 중 롤백 시 어떤 이벤트가 캡처되나요?
4. 이벤트: 이벤트 미발행 시 어떤 정책이 무력화되나요?
5. 상태: `체크인` 없이 `체크아웃`이 가능하도록 열어둔 부분이 있나요?

## 사용한 질문
- [x] 질문 3번 사용: "트랜잭션 롤백 시 BookingConfirmed 이벤트가 이미 발행됐으면 어떻게 되나요?"
  - 결과: eventual consistency 관리 필요성 인식, 보상 트랜잭션 패턴 검토
- [x] 질문 5번 사용: "체크인 없이 체크아웃 API가 호출되면?"
  - 결과: StayProgress에 상태 전이 가드 추가

## 체크리스트
- [x] 위 질문 중 최소 2개 이상을 사용했다.
- [x] 질문이 시나리오/테스트 결과를 참고하고 있다.
- [x] 정책/불변조건 중 어느 쪽을 겨냥했는지 명시했다.
