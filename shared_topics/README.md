# Shared Topics Guide

주차별로 반드시 확인해야 할 공통 목표/제약/토의 거리입니다. 세션을 시작하기 전에 해당 주차 파일(step1_requirements.md 등)을 읽고 체크리스트를 확인한 뒤 질문을 준비하세요.

- **Week1**: 용어/컨텍스트 정리, 요구사항 카드 제출 기준, 열린 질문 수집, PR 제출.
- **Week2**: 애그리게잇/이벤트 연결, 시나리오/테스트 최소 기준, 위험 케이스 논의, PR 제출.
- **Week3**: 질문 루프 통합, 트레이드오프 문서화, 최종 PR 제출.

AI도 이 폴더를 참조해 주차 목표를 상기시키므로, 참가자도 여기 내용을 기준으로 스스로 점검하면 됩니다.
