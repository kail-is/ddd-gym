# 2시간 오프라인 운영안
- 목적: 비동기 PR을 비교하고 최적 설계를 선택/결합한다.
- 원칙: 2시간 동안 새로운 문서 작성 금지, 토론만 진행.

## 시간표
1. **20분 – PR 선택**
   - 가장 토론 가치가 높은 PR 2~3개 선정.
   - 기준: 폭 넓은 트레이드오프, 경계/트랜잭션 논쟁거리.
2. **60분 – 설계 비교**
   - 각 PR 발표 10분 이내.
   - AI 질문 결과와 실제 수정 내역 비교.
   - 공통/차이/충돌 포인트를 화이트보드로 정리.
3. **30분 – 선택/정리**
   - 채택할 설계 조합을 결정하고 책임자 지정.
   - 후속 실험/리팩터링 계획 수립.
4. **10분 – 회고**
   - 배운 점, 다음 실험 아이디어 공유.

## 체크리스트
- [ ] 토론 대상 PR 링크 사전 공유.
- [ ] 발표자는 요구사항~테스트까지 5분 요약을 준비.
- [ ] 질문 전용 타이머를 두어 답변 독점 방지.
- [ ] 회고 내용은 노션/위키에 5줄 이상 요약.

## 팁
- 토론 중 실시간으로 의사결정 기록자를 한 명 지정한다.
- 합의가 안 되면 "다음 실험으로 보류" 라벨만 붙이고, 비동기 팔로업 PR을 만든다.
